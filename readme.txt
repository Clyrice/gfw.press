

欢迎使用 GFW.Press 新一代军用级高强度加密抗干扰网络数据高速传输软件


使用翻墙大杀器只需两步：


第一步、安装服务器

创建一个 CentOS8 服务器

https://www.vultr.com/?ref=7115062

一键安装

yum install wget -y -q ; wget -q -O gfw.sh https://raw.githubusercontent.com/chinashiyu/gfw.press/master/vultr_centos8.sh.txt ; sh gfw.sh


第二步、安装客户端

Windows版下载: 
https://gfw.press/GFW.Press.msi

Android版下载: 
https://gfw.press/GFW.Press.apk

全部版本下载: 
https://gfw.press/download.php


安装完成后，填写好服务器IP、端口和密码，就可以了

在 Windows 和部分 Android  手机，需要手工设置代理，地址为：127.0.0.1，端口为 3128

更多帮助请看： https://gfw.press/blog/?p=2047
